# Color Scheme Generator 🎨

A color palette generator built with vanilla JavaScript.
Pick a seed color, choose a color scheme mode, and get a beautiful palette instantly.

## Features

- Pick any seed color
- Choose from 5 color scheme modes (Monochrome, Complement, Analogic, Triad, Quad)
- Click any color to copy its HEX value to clipboard
- Smooth hover animations

## Built With

- HTML
- CSS
- JavaScript
- [The Color API](https://www.thecolorapi.com/)
